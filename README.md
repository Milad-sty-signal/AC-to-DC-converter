# AC to DC Converter – Full Wave Bridge & Transformer

## Overview
This project demonstrates the design and simulation of a linear AC–DC power supply
using a transformer, full-wave bridge rectifier, smoothing capacitor, and linear regulation.

## Features
- 50/60 Hz transformer step-down
- Full-wave bridge rectifier
- Capacitive filtering
- LTspice simulation
- Output ripple analysis

## Tools Used
- LTspice XVII

## Files
- `/ltspice` → LTspice schematic and models
- `/schematics` → circuit diagrams
- `/simulation_results` → output voltage and ripple plots

## How to Run
1. Open LTspice
2. Load `ac_dc_converter.asc`
3. Run transient analysis
